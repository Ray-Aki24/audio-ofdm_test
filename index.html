v2での改善点
✓ スペクトル包絡整合：チャイム音源の6〜8kHz帯振幅に合わせてOFDMサブキャリア振幅を調整。ホワイトノイズ感を解消。
✓ 帯域除去FIRフィルタ：チャイム信号の6〜8kHz成分を除去してからOFDMを重畳。干渉を低減。
✓ BPFによる受信帯域分離：受信時に6〜8kHz帯のみを抽出してから復調。SNRを大幅改善。
✓ 差動検波ロジック修正：D-BPSKの位相差計算を正確に実装。
✓ ガード時間拡大（512smp≒11.6ms）：室内残響への耐性を向上。
✓ 受信バッファ・プリアンブル検出改善：ScriptProcessorによるフレーム単位処理。
